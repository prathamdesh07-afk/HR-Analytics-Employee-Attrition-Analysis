# HR-Analytics-Employee-Attrition-Analysis
Exploratory Data Analysis of employee data to identify key factors influencing employee attrition using Python.

## Project Overview
This project analyzes employee data to identify factors contributing to employee attrition. Using Python and data visualization techniques, we explore patterns in employee satisfaction, workload, department distribution, and salary levels.

## Dataset
The dataset contains employee information such as:
- Satisfaction level
- Last evaluation
- Number of projects
- Average monthly hours
- Time spent at company
- Work accidents
- Promotion in last 5 years
- Department
- Salary level
- Employee attrition (left)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis Performed
- Data cleaning and preprocessing
- Duplicate record removal
- Exploratory Data Analysis (EDA)
- Employee attrition visualization
- Department-wise analysis
- Workload and satisfaction analysis

## Key Insights
- Employees with lower satisfaction levels are more likely to leave.
- Employees working extreme monthly hours show higher attrition.
- Salary level influences employee retention.
- Some departments show higher attrition patterns.

## Conclusion
HR analytics can help organizations identify the reasons behind employee turnover and design strategies to improve employee satisfaction and retention.
